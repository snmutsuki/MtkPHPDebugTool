###### mtk debug工具v1.0.0-beta

* 使用步骤：

  * 1、将本文件夹复制到服务器的根目录下，或者设置服务器的根目录为本文件夹

  * 2、输入网址locaohost:port//mtkDebug/mtk.php
  * 3、按照弹出的页面进行配置
  * 4、在远端项目中插入插件
  * 5、再次进入locaohost:port//mtkDebug/mtk.php
